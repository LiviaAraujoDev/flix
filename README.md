# Flix
O flix é um projeto que exibir um filme aleatório, por meio dos dados fornecidos pela API do themoviedb.org<br>

## Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- HTML
- CSS
- SASS
- JavaScript
- React JS<br><br>
## Run Locally

~~~bash
$ npm run dev
~~~
